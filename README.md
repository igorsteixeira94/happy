## Como Usar

```shell
git clone git@github.com:igorsteixeira94/happy.git
cd happy

#Iniciando o Web localhost:3000
cd web
yarn install
yarn start

#Iniciando o BackEnd localhost:3333
cd backend
yarn install
yarn dev

#Iniciando o Mobile 
cd mobile
yarn start
yarn android
```
